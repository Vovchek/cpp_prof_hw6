Detailed Description

template<typename V, V def_val>
class SparseMatrix< V, def_val >

SparseMatrix container class that stores only non-default cell values of a huge (INT_MAX x INT_MAX size) 2D matrix.

Template Parameters
Vcell type.
def_valdefault value for cells.

SparseMatrix stores pairs in std::map container with the key equal to the matrix row index and data storing corresponding SparseVector object.
When default value is written to the cell so the SparseVector object for this row becomes empty, then the corresponding element should be erased from the map.
In contrary, when non-default value is written to a cell of an empty SparseVector, then new row is inserted into map.
Row index ranges from 0 to MAX_INT.

Bug:
It is assumed that bracket operators after matrix variable always go in pair to access cell value, like: v2 = mx[i][j] = v;
If we'll need to access a row like r = mx[i] or mx[i] = r, we should implement corresponding operators - now it does not compile.
