Detailed Description

template<typename T, typename V, V def_val>
class Proxy< T, V, def_val >

Proxy for SparseVector & SparseMatrix objects to discer cell write or read.

Template Parameters
Tdata type - may be SparseVector<V,def_val> or V
Vcell type
def_valdefault value for cells of type V

SparseVector returns this Proxy when operator [] is envoked.
If the caller needs write access (like v[i] = value) it employes operator = ,
otherwise (like var = v[i]) const typecast V() operator reurns cell value (or default).

Indexing SparseMatrix to get a row.

Parameters
iRow index.

This operator is for addressing SparseMatrix rows. Because we do not want to create a real SparseVector object for this purpose, we simply return Proxy for SparseVector. Map element with key == i may be created if nessesery; then if it stays (or becomes) empty it will be erased in a Proxy destructor.
